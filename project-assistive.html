Assistive Web Navigation Device

The Idea

This project was developed to empower differently-abled individuals to browse the internet and navigate their computers without relying on traditional keyboards and mice. It specifically addresses challenges faced by individuals missing a hand, fingers, or an arm by offering a fully customizable solution tailored to their needs.

The current prototype is designed for individuals without hands, allowing them to control the mouse cursor using head movements. Additional functionalities, such as clicking, voice typing, and keyboard input, are integrated through touch sensor buttons on a wearable vest (a modified t-shirt). Built during a hackathon, the prototype may appear rushed, but it laid the foundation for further development. Some head-tracking technology and code were adapted from a previous immersive game controller project.

How It Works

The device consists of a headband and a vest. The headband tracks head movements using a built-in gyroscope and accelerometer, with backend software translating these movements into mouse cursor control.

The vest features customizable buttons that can be tailored to user preferences. In the demonstration, the preset controls included:

  • Activating voice typing using a Windows macro shortcut
  • A mouse left click
  • An enter key function
  • A button to stop the cursor from moving if needed

    • One major drawback of the current prototype is its reliance on wired connections, which limits flexibility and ease of use.

    What's Next

    Since this project was created for a hackathon, development paused once the event ended. However, future improvements include designing a more refined vest and transitioning the system to a fully wireless solution for greater usability.
